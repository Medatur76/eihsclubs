# EIHS Clubs
The website for all clubs at Energy Institute Highschool. Upkept by the Coding Club.
## Sites
* [Main](https://eihsclubs.com): General overview of EIHS's clubs
* [Outlet](https://outlet.eihsclubs.com): The magazine club at EIHS
* [Coding](https://coding.eihsclubs.com): The coding club (Kinda up but not really)
* [Vex](https://vex.eihsclubs.com): The VEX robots club
## Code features
* Built for ARM64 Linux architecture
* Made with a C framework
* Utilizes ASM for small features
* Automatic pulling from Git for live updates
* HTTP Method parsing
* Blocks requests that point outside of the ``web/`` folder
* Multi-domain handling from one socket
